## Description

Little login system that provides a jwt for valid logins Work in progress

You can run auth-gateway, auth-registration, auth-connector, auth-queue at the same time to make it work

Soon I will share the sample env

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
